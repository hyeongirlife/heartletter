# HeartLetter (하트레터)

연애편지 초안 작성을 도와주는 웹 애플리케이션입니다.

## 기능

- 맞춤형 연애편지 초안 생성
- 다양한 스타일과 분위기의 편지 템플릿
- 편지 저장 및 관리
- 다운로드 및 공유 기능

## 설치 및 실행

```bash
# 저장소 복제
git clone https://github.com/hyeongirlife/heartletter.git
cd heartletter

# 의존성 설치
npm install

# 개발 서버 실행
npm start
```

## 기술 스택

- React.js
- React Router
- Styled Components
- React Bootstrap
- React Icons
